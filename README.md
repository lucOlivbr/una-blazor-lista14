## Como rodar

1. Clone o repositório:
   git clone https://github.com/lucOlivbr/una-blazor-lista14.git

2. Acesse a pasta:
   cd una-blazor-lista14

3. Rode o projeto:
   dotnet run

4. Acesse no navegador: http://localhost:SEUPORT

## Tecnologia

Blazor WebAssembly com .NET 10 e Bootstrap.
